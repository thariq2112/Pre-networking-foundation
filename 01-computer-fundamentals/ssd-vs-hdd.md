# 6. SSD vs HDD

## a) What is an SSD?

An **SSD (Solid State Drive)** is a permanent storage device that stores data using **flash memory chips**.

Unlike an HDD, an SSD has **no moving parts**.

Because there are no moving parts, SSDs are much faster, quieter, and more durable.

An SSD stores:

- Operating System
- Applications
- Games
- Documents
- Photos
- Videos

The data remains stored even after the computer is turned off.

---

## b) What is an HDD?

An **HDD (Hard Disk Drive)** is a permanent storage device that stores data on **spinning magnetic disks called platters**.

A mechanical arm with a read/write head moves across the platters to read or write data.

Because the disks must spin and the arm must move, HDDs are slower than SSDs.

Like SSDs, HDDs also store:

- Operating System
- Applications
- Documents
- Photos
- Videos

The data remains stored even after the computer is turned off.

---

## c) Why are SSDs faster than HDDs?

The main reason is that SSDs have **no moving parts**.

When data is needed:

- An SSD accesses flash memory electronically.
- An HDD must spin its platters and move its read/write head to the correct location.

Since electronic access is much faster than mechanical movement, SSDs provide much quicker performance.

---

## d) How does an HDD work?

When you open a file stored on an HDD:

1. The disk platters begin spinning.
2. The read/write head moves to the correct location.
3. The required data is read from the magnetic surface.
4. The operating system loads the data into RAM.
5. The CPU begins executing the program.

This mechanical movement takes time.

---

## e) How does an SSD work?

When you open a file stored on an SSD:

1. The SSD controller locates the required data.
2. The flash memory chips send the data electronically.
3. The operating system loads the data into RAM.
4. The CPU begins executing the program.

Since there are no moving parts, this process is much faster.

---

## f) SSD vs HDD

| SSD | HDD |
|------|------|
| Uses flash memory | Uses magnetic spinning disks |
| No moving parts | Moving mechanical parts |
| Much faster | Slower |
| Silent | Produces noise while spinning |
| More durable | More sensitive to physical shocks |
| More expensive per GB | Cheaper per GB |
| Lower power consumption | Higher power consumption |

---

## g) Real-World Example

Imagine two libraries.

### HDD

A librarian must:

- Walk to a shelf.
- Find the correct book.
- Bring it back to you.

This takes time.

### SSD

Instead of walking, the book instantly appears on your desk.

No searching.

No movement.

This is similar to how an SSD retrieves data electronically.

---

## h) Where are SSDs and HDDs used?

### SSDs are commonly used in:

- Modern laptops
- Gaming PCs
- High-performance desktops
- Servers
- Workstations

### HDDs are commonly used in:

- Backup storage
- NAS (Network Attached Storage)
- Security camera recording systems
- Large media storage
- Archive servers

- Access speed decreases.
- Cost per GB generally decreases.

---

## Conclusion: SSD vs HDD

Both SSDs and HDDs are **non-volatile storage devices**, meaning they keep data even after the computer is turned off.

An SSD stores data using flash memory and has no moving parts, making it much faster, quieter, and more durable.

An HDD stores data on spinning magnetic disks and uses mechanical components, making it slower but more affordable for large amounts of storage.

For everyday computing, SSDs provide the best performance, while HDDs are still useful for storing large amounts of data at a lower cost.

---

## Common Misconceptions

❌ SSD is a type of RAM.

✔️ False. SSD is a permanent storage device, while RAM is temporary memory.

---

❌ HDD stores data in RAM.

✔️ False. HDD stores data on magnetic platters. The operating system loads required data from the HDD into RAM.

---

❌ SSDs never fail because they have no moving parts.

✔️ False. SSDs are generally more reliable than HDDs, but they can still fail due to wear, controller issues, or other hardware problems.

---

❌ HDDs are obsolete.

✔️ False. HDDs are still widely used for backups and large-capacity storage because they are much cheaper per GB.
